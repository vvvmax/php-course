התקנת שרת לוקאלי
===
PHP זה שפת שרת. כלומר כל הקוד שלו מעובד קודם כל על השרת ורק אחר כך מוחזר לצד לקוח בעזרת HTML, XML או תמונה (כל התוכן שהBROWSER מבין). 
זה אומר שקובץ PHP רץ רק בסביבה שבה מותקנת תוכנת מיוחדת שיכולה להריץ PHP. 
יש 2 אפשרויות. 
1. להעלות את הקובץ לשרת מרוחק (REMOTE SERVER) ולהריץ אותו שם.
2. להתקין תוכנה בשם שרת לוקאלי במחשב אישי ולהריץ את הקובץ שם. 

האופציה הראשונה נוחה כדי להראות את התוצאה לאחרים, או לפרסם את האתר או את האפליקציה שעשיתם). 
האופציה השניה טובה בשביל הפיתוח. בקורס חשוב שתשתמשו בשני האופציות.

סביבת PHP
---

בשביל שהPHP ירוץ צריכים להיות מותקנים הרכיבים הבאים: 
1. תוכנת השרת - Apache Server, IIS, Nginx
2. רכיב PHP שמתחבר לשרת. 
3. רכיב בסיס נתונים (MySql במקרה שלנו) שמתחבר לשרת. 
כמובן ששלושתם צריכים להיות מקונפגים כראוי כדי שיעבדו ביחד ויכללו את כל הדרוש.

למזלינו קיימים חבילות מוכנות המקונפגות מראש המכילות את שלושת הרכיבים ההופכות הת התקנת השרת הלוקאלי לפשוט ביותר.
החביתה המפורסמת ביותר היא XAMPP. 

התקנת XAMPP
---

1. הולכים לכתובת זו: https://www.apachefriends.org/download.html ומורידים את הגרסה האחרונה של XAMPP (עם PHP 7). 
![הורדת XAMPP מהאתר](http://valianocode.com/phpcourse/intro/install_xampp1.png "הורדת XAMPP מהאתר")

2. מתקינים את התוכנה. עם סיות ההתקנה תופיע תיקיית XAMPP. 

** אם במהלך ההתקנה תידרש להסכים להערות של חומת אש או של אנטיוירוס, תסכים (allow). 

![מתקינים XAMPP](http://valianocode.com/phpcourse/intro/install_xampp2.png "מתקינים XAMPP")
